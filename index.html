Introdução ao HTML na Prática

Estrutura Básica do HTML

A Estrutura básica do HTML é dividida basicamente em 3 partes. A primeira iniciando com a tag html que permite que seja utilizado as tags e outras coisas para criar um website, depois são dividas em head e body, o head é a cabeça do código onde ficam as configurações do site e o body é como se fosse o corpo onde tudo que escrevemos dentro da tag body vai aparecer no navegador.

Falando sobre Tags

As tags são como etiquetas, elas demarcam que tal espação vai ter tal função.

Ex: a tag p é utilizada pra parágrafos, textos.

Obs: As tags possiem atributos, podendo colocar uma identificação (id, class), ou até mesmo alterar a forma, tamanho ou função.

Textos

h1 até h6: Tag para titulos e subtitulos

p: Tag para parágrafos

mark: Tag para marcação, citações

small: Tag para textos pequenos

i: Tag para deixar o texto em italico

u: Tag para colocar um linha embaixo

strong: Tag para deixar o texto em negrito

ol: Tag para criar uma lista ordenada

ul: Tag para criar uma lista não ordenada

li: Tag para criar os itens de uma lista

a: Tag para criar um link

hr: Tag para criar um linha na horizontal pra separar

sub: Tag para deixar o item acima da palavra

sup: Tag para deixar o item abaixo da palavra

blockquote: Tag para definir com bloco de texto
