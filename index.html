O que são Tags

As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing). O fechamento de uma tag será definido com uma barra (/), sendo que no caso das tags de auto fechamento, não há necessidade da presença desse caractere.

Tags básicas

h1 até h6

São as tags utilizadas para titulos e subtitulos.

p

É utilizada para paragrafos.

mark

É usada para definir o texto marcado. É usado para destacar a parte do texto em um parágrafo.

small

É utilizada para definir um texto menor

i

É utilizadas para deixar a/o palavra/texto em italico

u

É utilizadas para deixar a/o palavra/texto em negrito

strong

É utilizada para representar algum texto não articulado e com estilo diferente do texto normal

ol

É usada para criar uma lista ordenada

ul

É usada para criar uma lista não ordenada

li

É usada para criar os itens de uma lista, tanto para a ordenada quanto para a não ordenada

a

Serve para criar/definir uma palavra ou texto como link, torna-lo clicavel

hr

Cria uma linha na horizontal

sub

Define o texto subscrito. Ele aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor

sup

Define o texto sobrescrito. Ele aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor

blockquote

Indica que o texto incluído é uma citação estendida. Normalmente, isso é renderizado visualmente por recuo

Outras Tags

font

Foi usado em HTML 4 para especificar a face da fonte, o tamanho da fonte e a cor do texto

del

Representa uma parte do texto que foi excluída de um documento

q

Indica uma pequena citação. Este elemento destina-se a citações curtas que não requerem marcações de parágrafo; diferente do blockquote que serve para citações maiores use o elemento

abbr

Define as informações de contato do autor/proprietário de um documento ou artigo. As informações de contato podem ser um endereço de e-mail, URL, endereço físico, número de telefone, identificador de mídia social, etc.
